# Presime ⏱️

Presime is a modern, beautifully designed productivity and focus timer application built for Android. It helps users manage their time effectively using focused sessions, seamless progress tracking, and intuitive statistics.

## ✨ Features

- **Focus Timers**: High-quality stopwatch interfaces with pulsing animations during active focus sessions.
- **Real-Time Statistics**: Comprehensive session tracking and productivity charts using [Vico Charts](https://patrykandpatrick.com/vico/).
- **Modern App Design**: Polished, edge-to-edge dark theme UI built completely with Jetpack Compose and Material 3.
- **Reliable Persistence**: Uses Room Database for robust session storage and DataStore for user preferences.
- **Smooth Navigation**: Simple and intuitive navigation flows built with Jetpack Navigation Compose.

## 🛠️ Tech Stack

Presime is built adhering to modern Android development practices:

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Toolkit**: [Jetpack Compose](https://developer.android.com/jetpack/compose) & Material 3
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Injection**: [Dagger-Hilt](https://dagger.dev/hilt/)
- **Local Storage**: [Room](https://developer.android.com/training/data-storage/room) & [DataStore](https://developer.android.com/topic/libraries/architecture/datastore)
- **Navigation**: Jetpack Navigation Compose
- **Data Visualization**: Vico Compose Charts

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest stable version recommended)
- Android SDK version 26 (Android 8.0) or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/twenty1-store/Presime.git
   ```
2. Open the project in Android Studio.
3. Allow Gradle to sync the required dependencies.
4. Run the application on an emulator or physical device (Target SDK 36 is supported).
